# Pixelgame

Play Dev - trilha de aprendizado gamificada.

## Requisitos

- Node.js (recomendado: LTS)
- MongoDB (local ou remoto)

## Como rodar

1. Instale as dependências:

```bash
npm install
```

2. Configure as variáveis de ambiente:

```bash
cp .env.example .env
```

Edite o arquivo `.env` conforme necessário.

3. Inicie o servidor:

```bash
npm start
```

Para desenvolvimento (com reload):

```bash
npm run dev
```

## Arquivos úteis

- `index.html`: tela principal
- `demo.html`: demo/variante
- `server.js`: API/servidor Express
